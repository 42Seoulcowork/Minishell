# minishell

## 1. git 관련

- 별일 없으면 **무조건 한 파일씩!** commit 하기
- log 메세지는 다음과 같이 할 것
```shell
{변경한 파일 이름} | {변경한 내용}
```

- push하면 바로 slack으로 알려주기
- slack보고 먼저 pull하는거 잊지 말기
- 만약 충돌하면 slack
